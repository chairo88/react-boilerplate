# react-podcaster

## Environment setup

 - Install [Node.js](https://nodejs.org/)
 - Run `npm install` to install the project dependencies

## Development

### 1) Run and build the app:

```
npm run start or npm start
```
This command runs the app on port 3000

### 4) Navigate to:
```
http://localhost:3000
```

## Debug

### 1) Build the assets:

```
npm run build
```
